# Case Study: Cal.com Analysis

Analyse visuelle et ergonomique de Cal.com pour servir de référence technique.

## 1. UI Analysis
Détails extraits du système Cal.com :
- **Palette :** Utilisation intensive du Noir pur (#000000) et des nuances de Zinc.
- **Typographie :** Mariage de `Cal Sans` (Bold/Semi-bold) pour le branding et `Inter` pour l'interface.
- **Radii :** Bordures arrondies systématiques de **12px** pour les boutons et **16-24px** pour les conteneurs.

## 2. UX Analysis
- **Principle: Calm Power** : L'interface cache la puissance technique derrière un minimalisme radical.
- **Zero-Friction** : Optimisation des flux pour réduire la charge cognitive (ex: Timezone handling).
