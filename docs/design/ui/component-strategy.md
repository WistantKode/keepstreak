# Component Strategy

## 1. Foundation Components (Cal.com)
Nous utilisons les primitives de Cal.com pour garantir la robustesse :
- **Navigation :** Sidebar, Breadcrumbs, Profile menus.
- **Inputs :** Forms, Buttons, Switches (Brand Blue).
- **Layout :** Grids et Shells responsives.

## 2. Signature Components (Cyber-Premium)
Composants spécialisés créés pour KeepStreak :
- **PulseDisplay :** Visualisation animée de la santé du worker.
- **HandshakeTheater :** Séquence d'onboarding "magique".
- **StealthLogs :** Terminal haute-fidélité pour les événements d'orchestration.
- **SecurityBadge :** Statut de vérification HMAC interactif.
