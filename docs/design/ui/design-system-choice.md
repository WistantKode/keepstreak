# Design System Choice: Cal.com

## 1. Pourquoi Cal.com ?
**Cal.com** a été choisi comme fondation pour KeepStreak en raison de son excellence dans le domaine des outils SaaS orientés développeurs.

### Justification
- **Fiabilité Éprouvée :** Une référence pour les interfaces propres et fiables.
- **Synergie Immédiate :** Code source disponible pour une fidélité maximale.
- **Ergonomie "Dev" :** Parfaitement adapté à notre audience technique.

## 2. Stratégie de Personnalisation
- **Extension :** Ajout de composants spécifiques pour la visualisation HMAC et les logs Redis.
- **Couche Motion :** Injection de `framer-motion` (Inspiration Framer) pour l'effet "Wow" et la fluidité des handshakes.
